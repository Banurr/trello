# Trello Clone

## Description
Trello Clone is a project inspired by Trello, designed to manage tasks with features for task classification, development tracking, and status checking. It utilizes Spring Boot for backend development, PostgreSQL for data storage, and includes CRUD operations for folders, tasks, and status management.

## Features
- **Task Management:** Create, update, and delete tasks.
- **Task Classification:** Organize tasks into folders or categories.
- **Task Development:** Track the progress and development stages of tasks.
- **Status Management:** Change and update task statuses (e.g., to-do, in progress, done).

## Technologies Used
- Spring Boot
- PostgreSQL
- Spring Data JPA
- Hibernate
- CRUD Operations
